## Download Instructions

### USGS Images
1. Click the direct S3 URLs above
2. Right-click → Save As
3. For full resolution: remove `/styles/half_width/public/` or `/styles/full_width/public/` from the URL
4. No attribution needed (public domain)

### Unsplash / Pexels / Pixabay
1. Visit the search page URLs above
2. Click on images you want
3. Click "Download" or "Free Download"
4. Select resolution (use highest available)
5. No attribution needed

### Wikimedia Commons
1. Visit category pages above
2. Click individual images
3. Check license (look for CC0 or CC-BY)
4. Download from file page
5. Provide attribution if required by license

## Notes

- **Optimize all images** before using on the website — use Next.js Image component with WebP format
- **Alt text matters for SEO** — describe volcanic features specifically: "Kilauea lava fountain erupting at night" not just "volcano"
- **Dark overlay technique** — many of these images work best with a 40-60% black overlay to maintain readability of white text
- **USGS is your best source** — 2,986 images, all public domain, many are CURRENT (2024-2026 eruption), and they're scientifically legitimate which adds credibility to your volcano positioning
